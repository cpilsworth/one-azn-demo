# Deutsche S3-Leitlinie SLE veröffentlicht

**Systemischer Lupus erythematodes (SLE): Erste deutsche Leitlinie veröffentlicht**

Mit der Herausgabe der S3-Leitlinie zum Management des systemischen Lupus erythematodes (SLE)<sup>1</sup> steht nun zum ersten Mal eine deutsche Guideline zur Therapie des SLE zur Verfügung. Die S3-Leitlinie orientiert sich unter anderem an den Empfehlungen der EULAR (European Alliance of Associations for Rheumatology).<sup>2</sup> Sie behandelt dabei das übergeordnete Therapieziel Remission – das besonders in den Vordergrund gestellt wird – als zentrales Thema im SLE-Therapiemanagement, sowie wichtige Informationen über Glucocorticoide als Überbrückungstherapie, die Gabe von Hydroxychloroquin für alle Patient:innen und der zeitnahen Einleitung einer immunsuppressiven oder Biologika-Therapie bei Nichtansprechen (siehe Abb.1).<sup>1</sup>

+---------------------------------------------------------------------------------------------------------------+
| **Accordion-Section**                                                                                         |
+-----------------------------------------------+---------------------------------------------------------------+
| Biologika als Standardtherapie bei SLE        | Laut der aktuellen S3-Leitlinie spielen Biologika eine        |
|                                               | zentrale Rolle im Therapiemanagement von SLE. Unmittelbar     |
|                                               | nach Diagnosestellung sollte bei allen Patient:innen eine     |
|                                               | Therapie mit Hydroxychloroquin (HCQ) begonnen werden.         |
|                                               | Reichen HCQ und maximal 5 mg Prednisolon-Äquivalent pro Tag   |
|                                               | zur Kontrolle der Krankheitsaktivität nicht aus, sollte       |
|                                               | frühzeitig eine Behandlung mit Biologika wie Anifrolumab      |
|                                               | oder eine immunmodulierende oder immunsuppressive Therapie    |
|                                               | eingeleitet werden (siehe Abb. 1).                            |
+-----------------------------------------------+---------------------------------------------------------------+
| Übergeordnetes Therapieziel Remission         | Daten der Beobachtungsstudie SPOCS zeigen deutlich, dass      |
|                                               | bisher wenige Patient:innen mit SLE in Deutschland eine       |
|                                               | DORIS-Remission erreichen: Im Schnitt befanden sich nur       |
|                                               | 14 % der SLE-Patient:innen nach 36 Monaten Therapie in        |
|                                               | Remission bevor Anifrolumab als Therapieoption verfügbar war. |
+-----------------------------------------------+---------------------------------------------------------------+
| Die Initialtherapie: Hydroxychloroquin        | Nach der Diagnose des SLE besteht die Basistherapie aus HCQ,  |
|                                               | das alle Patient:innen erhalten sollen (Empfehlungsgrad A,    |
|                                               | Evidenzlevel 1). Die tägliche HCQ-Dosis sollte bei 5 mg/kg    |
|                                               | Körpergewicht liegen.                                         |
+-----------------------------------------------+---------------------------------------------------------------+
| Glucocorticoide als Überbrückungstherapie     | Die Dosierung von Glucocorticoiden (GC) sollte sich an Art    |
|                                               | und Ausprägung der Organbeteiligung orientieren. Idealerweise |
|                                               | sollte ein vollständiges Absetzen der GC-Therapie angestrebt  |
|                                               | werden, sonst sollte eine rasche Reduktion auf eine           |
|                                               | Erhaltungsdosis von ≤ 5 mg/Tag Prednisolon-Äquivalent         |
|                                               | erfolgen.                                                     |
+-----------------------------------------------+---------------------------------------------------------------+
| Fatigue bei SLE                               | Fatigue ist für viele Patient:innen ein führendes Symptom     |
|                                               | ihrer Erkrankung. Zur Behandlung sollten Patient:innen-       |
|                                               | Schulungen und körperliches Training durchgeführt werden.     |
|                                               | Die Leitlinie fasst zusammen, dass konsequente körperliche    |
|                                               | Aktivität die aerobe und die funktionelle Kapazität           |
|                                               | verbessert und Fatigue und depressive Symptome bei SLE        |
|                                               | vermindern kann.                                              |
+-----------------------------------------------+---------------------------------------------------------------+

---

*DORIS-Remission: klinischer SLEDAI-2K-Gesamtscore = 0; ärztliche Gesamtbewertung (0–3) < 0,5; Prednison/äquivalente Dosis ≤ 5 mg/Tag.

°ST: Antimalariamittel, Immunsuppressiva, nichtsteroidale Antirheumatika und Glucocorticoide.
